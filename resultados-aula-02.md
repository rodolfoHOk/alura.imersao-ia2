def fibonacci(n):
  """
  Esta função calcula o n-ésimo número da sequência de Fibonacci.
  """
  if n == 0:
    return 0
  elif n == 1:
    return 1
  else:
    return fibonacci(n-1) + fibonacci(n-2)

# Exemplo de uso:
print(fibonacci(6))  # Saída: 8

Explicação:

- A função fibonacci(n) usa recursão para calcular o n-ésimo número de Fibonacci.
- Os casos base são definidos para 0 e 1, retornando 0 e 1 respectivamente.
- Para qualquer outro número n, a função se chama recursivamente duas vezes, calculando (n-1) e (n-2) da sequência, e então retorna a soma desses dois valores.

Exemplo de uso:

- Chamar fibonacci(6) retornará 8, pois o sexto número na sequência de Fibonacci é 8.

/**
 * Calcula o enésimo número de Fibonacci usando memorização.
 *
 * @param n - O índice do número de Fibonacci a ser calculado.
 * @returns O enésimo número de Fibonacci.
 */
function fibonacci(n: number): number {
  // Cria um cache para armazenar os valores de Fibonacci calculados.
  const cache: number[] = [];

  // Define uma função recursiva interna para calcular o enésimo número de Fibonacci.
  function fibonacciInterno(n: number): number {
    // Se o valor já estiver no cache, retorna-o diretamente.
    if (cache[n] !== undefined) {
      return cache[n];
    }

    // Caso base: o 0º e o 1º números de Fibonacci são 0 e 1, respectivamente.
    if (n <= 1) {
      return n;
    }

    // Calcula o enésimo número de Fibonacci recursivamente e armazena-o no cache.
    const resultado = fibonacciInterno(n - 1) + fibonacciInterno(n - 2);
    cache[n] = resultado;
    return resultado;
  }

  // Chama a função recursiva interna para iniciar o cálculo.
  return fibonacciInterno(n);
}

Explicação:

- Memorização: A função usa memorização para armazenar os valores de Fibonacci calculados em um cache. Isso evita cálculos redundantes de valores de Fibonacci que já foram calculados anteriormente, melhorando significativamente o desempenho.
- Recursão: A função usa recursão para calcular o enésimo número de Fibonacci em termos dos dois números anteriores. Isso é feito pela função fibonacciInterno.
- Caso base: Os casos base da recursão são quando n é 0 ou 1, onde os valores de Fibonacci são 0 e 1, respectivamente.
- Cache: O cache é um array que armazena os valores de Fibonacci calculados. O índice do array corresponde ao índice do número de Fibonacci e o valor armazenado no índice é o valor de Fibonacci.
- Complexidade de tempo: Com memorização, a complexidade de tempo desta função é O(n), pois cada número de Fibonacci é calculado no máximo uma vez. Sem memorização, a complexidade de tempo seria exponencial, o que a tornaria muito ineficiente para valores grandes de n.

Como usar a função:

const quintoFibonacci = fibonacci(5); // quintoFibonacci será 5
const decimoFibonacci = fibonacci(10); // decimoFibonacci será 55
